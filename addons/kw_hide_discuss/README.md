# Hide discuss

[![Maintainer: Kitworks](https://img.shields.io/badge/maintainer-Kitworks-purple.png)](https://kitworks.systems/)
[![Docs: Hide_Discuss](https://img.shields.io/badge/docs-Hide_Discuss-yellowgreen.png)](https://kitworks.systems/)

This module allows restricting access to the Discuss menu in Odoo by hiding it from users who are not in a specific security group.

## Features

- Creates a new access rights category "Discuss access"
- Adds a new security group "Discuss user"
- Restricts visibility of the Discuss menu only to users in the "Discuss user" group
- Allows system administrators to control who has access to discussion features

## Usage

1. Install the module
2. Add users who need access to the Discuss menu to the "Discuss user" group
3. Users who are not in this group will not see the Discuss menu in the Odoo main menu
