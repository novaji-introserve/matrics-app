Hide Discuss Menu


This module enhances Odoo's interface customization by providing granular control over the Discuss menu visibility. It allows administrators to restrict access to the Discuss feature through user groups, helping to streamline the user interface and enhance security without affecting other messaging functionality.

Key features include:

  • Group-Based Control: Manage Discuss menu visibility through user groups
  • Clean Interface: Remove unnecessary menu items for non-authorized users
  • Simple Management: Easy user group administration
  • Full Compatibility: Works seamlessly with all Odoo versions
  • Preserved Functionality: All other messaging features remain intact

Configuration

Setting up the module is straightforward:

  1. Install the module
  2. Navigate to Settings -> Users & Companies -> Groups
  3. Add users to the "Discuss User" group to grant them access

Benefits

  • Cleaner User Interface: Remove unnecessary menu items for specific users
  • Enhanced Security: Control access to messaging features
  • Simplified Experience: Users only see relevant menu items
  • Easy Administration: Manage access through standard Odoo groups
  • No Functionality Loss: Other messaging features remain fully operational
