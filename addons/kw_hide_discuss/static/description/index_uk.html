Приховування меню Обговорення


Цей модуль покращує налаштування інтерфейсу Odoo, надаючи детальний контроль над видимістю меню Обговорення. Він дозволяє адміністраторам обмежувати доступ до функції Обговорення через групи користувачів, допомагаючи оптимізувати інтерфейс користувача та підвищити безпеку без впливу на інші функції обміну повідомленнями.

Основні можливості:

  • Груповий контроль: Керування видимістю меню Обговорення через групи користувачів
  • Чистий інтерфейс: Видалення непотрібних пунктів меню для неавторизованих користувачів
  • Просте керування: Легке адміністрування груп користувачів
  • Повна сумісність: Працює безперешкодно з усіма версіями Odoo
  • Збереження функціоналу: Всі інші функції обміну повідомленнями залишаються незмінними

Налаштування

Налаштування модуля є простим:

  1. Встановіть модуль
  2. Перейдіть до Налаштування -> Користувачі та компанії -> Групи
  3. Додайте користувачів до групи "Користувач Обговорення" для надання їм доступу

Переваги

  • Чистіший інтерфейс: Видалення непотрібних пунктів меню для певних користувачів
  • Підвищена безпека: Контроль доступу до функцій обміну повідомленнями
  • Спрощений досвід: Користувачі бачать лише релевантні пункти меню
  • Легке адміністрування: Керування доступом через стандартні групи Odoo
  • Без втрати функціоналу: Інші функції обміну повідомленнями залишаються повністю робочими
